## Features
- City-based weather search
- Backend with Node.js + Express
- API key protection using `.env`
- Responsive frontend with HTML/CSS/JS

## Tech Stack
- Node js
- Express
- OpenWeatherMap API
- HTML, CSS, JavaScript
